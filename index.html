Code for Vision Azərbaycan texnologiya səhnəsinə dəstək olmaq məqsədi daşıyır

  • İnsanları birləşdirmək. Texnologiya istifadəçi qruplarının kəşf edilməsini və bir-birləri arasında ünsiyyəti təşviq edir.
  • Bilik paylaşmaq. Öyrənmə, mentor və paylaşma tövsiyələri. Yeni gələnlər və mütəxəssislər arasında texnoloji boşluğu azaldır.
  • Əməkdaşlıq. FOSS layihələri, icma qurma və kommersiya layihələri ilə birlikdə işləmək üçün həmfikir insanlar tapmaqa kömək edir.
